## Production Deployment Steps

### 1. Asset Preparation
Create the following image assets:
- favicon.ico (32x32)
- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png (180x180)

- og-image.png (1200x630)
- twitter-image.png (1200x630)
- logo.png (company logo)
- screenshot.png (product screenshot)

### 2. Server Configuration
```nginx
# Nginx example
server {
  listen 443 ssl http2;
  server_name www.originlytics.com;

  # SSL configuration
  ssl_certificate /path/to/cert.pem;
  ssl_certificate_key /path/to/key.pem;

  # Security headers
  add_header Strict-Transport-Security "max-age=31536000; includeSubDomains" always;
  add_header X-Content-Type-Options "nosniff" always;
  add_header X-Frame-Options "SAMEORIGIN" always;

  # Compression
  gzip on;
  gzip_types text/css application/javascript application/json;

  # Caching
  location ~* \.(css|js|jpg|png|svg|woff2)$ {
    expires 1y;
    add_header Cache-Control "public, immutable";
  }

  # Root
  root /var/www/originlytics;
  index index.html;
}

# Redirect HTTP to HTTPS
server {
  listen 80;
  server_name www.originlytics.com;
  return 301 https://$server_name$request_uri;
}
```

### 3. Analytics Setup
Add to `<head>` section (before closing `</head>`):
```html
<!-- Google Analytics 4 -->

  gtag('js', new Date());

</script>
```
